# JS Practice Log

This repository contains daily JavaScript exercises I’ve completed to improve my skills.  
It’s organized by topic and date, so you can see my progress over time.

## Structure

- **arrays/** – Exercises related to arrays
- **dom/** – Exercises manipulating the DOM
- **classes/** – Exercises using JavaScript classes
- **recursion/** – Recursive function exercises
- **etc./** – Other topics

Each exercise folder contains:
- `index.js` (or a descriptive JS file) – the code
- `README.md` – brief explanation of what the exercise does, what I practiced, and possible improvements

> Note: Complete projects are in separate repositories.
